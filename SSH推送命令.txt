# SSH推送命令 - 请手动在终端执行

## 当前状态
- SSH密钥已存在：~/.ssh/id_rsa
- 远程仓库已配置：git@github.com:weiyan08/talisman-generator.git
- 需要输入SSH密钥密码

## 请在终端中执行以下命令：

### 1. 启动SSH Agent
```bash
eval "$(ssh-agent -s)"
```

### 2. 添加SSH私钥（会提示输入密码）
```bash
ssh-add ~/.ssh/id_rsa
```
**此时需要输入您当初创建SSH密钥时设置的密码**

### 3. 测试SSH连接
```bash
ssh -T git@github.com
```
应该看到：Hi weiyan08! You've successfully authenticated, but GitHub does not provide shell access.

### 4. 推送代码到GitHub
```bash
git push origin main
```

## 如果遇到问题：

### 如果SSH密码错误或忘记：
```bash
# 1. 停止SSH Agent
ssh-agent -k

# 2. 删除现有密钥（可选）
rm ~/.ssh/id_rsa ~/.ssh/id_rsa.pub

# 3. 重新生成无密码的SSH密钥
ssh-keygen -t rsa -b 4096 -C "weiyan08@example.com"
# 提示时直接按回车，不设置密码

# 4. 将公钥添加到GitHub
cat ~/.ssh/id_rsa.pub
# 复制输出的公钥内容到GitHub设置

# 5. 重复上述步骤1-4
```

### 或者使用GitHub CLI（最简单）：
```bash
# 1. 安装GitHub CLI
brew install gh

# 2. 登录GitHub
gh auth login

# 3. 推送代码
git push origin main
```

## GitHub SSH密钥添加步骤：
1. 打开 https://github.com/settings/keys
2. 点击 "New SSH key"
3. 填写标题（如："MacBook Pro SSH Key"）
4. 粘贴 ~/.ssh/id_rsa.pub 的内容
5. 点击 "Add SSH key"
